# SaTScan Parameter Reference

A comprehensive reference for all SaTScan parameters available via
`epidscan`. Use these parameters in a `.prm` template file or as
overrides in `satscanr(...)`.

## Details

SaTScan is a software for spatial, temporal, and space-time scan
statistics. This documentation object serves as a placeholder for the
parameter reference.

For a full, searchable list of all parameters, please see the package
vignette:
[`vignette("satscan-parameters", package = "epidscan")`](https://Songyosr.github.io/epidscan/articles/satscan-parameters.md)

or visit the [SaTScan website](http://www.satscan.org/) for full
methodology.
