# New Mexico Lung Cancer Data (Population File)

Yearly population counts for New Mexico counties, 1973-1991.

## Usage

``` r
NMlung_pop
```

## Format

A data frame with 66,115 rows and 5 variables:

- county:

  County name (Location ID).

- year:

  Census year (Integer).

- population:

  Population count.

- age_group:

  Age group (18 levels: \<5, 5-9, ..., 85+).

- sex:

  Sex (Male, Female).
