# Print SaTScan Results

By default, prints the raw SaTScan text output if available. Use
`summary(x)` for a structured analytical summary.

## Usage

``` r
# S3 method for class 'satscan_result'
print(x, ...)
```

## Arguments

- x:

  A `satscan_result` object.

- ...:

  Additional arguments (currently unused).

## Value

Invisibly returns `x`.
