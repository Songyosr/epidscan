# New Mexico Lung Cancer Data (Coordinates)

Centroid coordinates for New Mexico counties (Cartesian).

## Usage

``` r
NMlung_geo
```

## Format

A data frame with 32 rows and 3 variables:

- county:

  County name (Location ID).

- x_km:

  X coordinate in km.

- y_km:

  Y coordinate in km.
