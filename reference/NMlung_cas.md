# New Mexico Lung Cancer Data (Case File)

Lung cancer cases in New Mexico counties, 1973-1991.

## Usage

``` r
NMlung_cas
```

## Format

A data frame with 7,793 rows and 5 variables:

- county:

  County name (Location ID).

- cases:

  Number of cases (1 for individual records).

- date:

  Date of diagnosis (Month/Year precision, day set to 1).

- age_group:

  Age group (18 levels: \<5, 5-9, ..., 85+).

- sex:

  Sex (Male, Female).
