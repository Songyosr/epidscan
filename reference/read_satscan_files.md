# Read SaTScan Output Files

Reads the output files (.col.dbf, .gis.dbf, etc.) generated by a SaTScan
run. This replaces `rsatscan`'s file reading logic.

## Usage

``` r
read_satscan_files(out_dir, project_name, verbose = FALSE)
```

## Arguments

- out_dir:

  Directory where output files are located.

- project_name:

  Base name of the project (e.g. "epid").

- verbose:

  Logical.

## Value

A list mimicking the structure of `rsatscan` output for compatibility.
