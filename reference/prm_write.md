# Write PRM List to File

Serializes a prm_list back to a PRM file using skeleton injection. The
original structure (comments, sections, order) is preserved.

## Usage

``` r
prm_write(prm, path)
```

## Arguments

- prm:

  A `prm_list` object.

- path:

  Output file path.

## Value

Invisibly returns the path.
